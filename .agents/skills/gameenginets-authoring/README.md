# GameEngineTS AI Authoring Skill

This package teaches coding agents how to build and modify games using GameEngineTS without treating the engine as one undifferentiated codebase.

It includes a compact `SKILL.md`, repository-wide `AGENTS.md`, focused references, starter templates, and a validation scaffold.

## Suggested use

Keep this folder in the repository while the authoring workflow evolves. For agents that support project skills, point the agent at this folder or copy it into the agent's project-skill directory. Keep `AGENTS.md` at the repository root when repository instructions are supported.

## Status

This is an initial authoring skill based on the audited GameEngineTS architecture. It intentionally avoids claiming that every legacy or low-memory subsystem is finalized.
